## Role Description

Responsible for managing Sagareus field asset inventory across all storage locations and deployments.

## Key Responsibilities

* **Asset checkout coordination** — all checkouts go through this role
* **Inventory tracking** — maintain [Asset Check Out](https://docs.google.com/spreadsheets/d/1n2k7oY9oGUadTmFZWtDe7DN47BAhI0L9YnF-Mri5K1U/edit?usp=sharing) spreadsheet and [Asset Photos](https://drive.google.com/drive/folders/1JP_Bve2NE3Ch-7H-sOifGHjd0eXgip_Y?usp=drive_link)
* **Asset reclamation** — track deployed assets and coordinate pickup during move-out via subtask assignment
* **New asset registration** — label, photograph, and log all new inventory
* **Storage facility setup** — coordinate new storage locations per [Asset Management](../section-indexes/sops-2-asset-management.md) requirements
* **Regional coordination** — work with [Senior Field Tech](senior-field-tech.md) and [Turn Over Coordinator Lead](turn-over-coordinator-lead.md) for storage management
