## Role Description

Owns turn over performance end-to-end. All turn over related issues escalate to this role. Manages the full turn over cycle from move-out inspection through rent-ready.

## Key Responsibilities

* Owns turn over performance and timeline
* All turn over related issues escalate here
* Vendor dispatching for rent ready
* Owner communications for turn over status
* Move-out inspection oversight
* Deposit return coordination (target 25 days)
* Security / property compromised response
* Locksmith coordination for key issues
* In-person verification of completed turn over work
* "Home Leased Out" coordination with vendors
* Mailbox rekey
