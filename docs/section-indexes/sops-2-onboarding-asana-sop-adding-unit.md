:::info
This variant covers **existing clients adding a unit to an existing property** — ADUs, DADUs, conversions, or when a unit the owner/family was occupying becomes available for rental. Use the **(UNIT) <Address> / <Owner Name>** task template in Client Relations 2.0.

:::

## When This Runs

Existing client adds a unit to an existing property, for example:

* Built an ADU or DADU on-site
* Converted a single-family home into multiple units
* A unit the owner or family member was occupying is now being vacated and leased

Because the property and owner already exist in Buildium and Asana, this variant has far fewer steps than the New Client or Adding Property flows. Utilities and vendors are already established.

## Prerequisites

* The existing PM Agreement must cover the additional unit (verify in step 1). If it does not, a PMA revision is required before this workflow.
* An existing parent task in Client Relations 2.0 for the property.
