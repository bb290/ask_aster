Pre Move-Out is the first phase of a Leasing 2.0 project. Its purpose is to set the engagement up for success — close out the outgoing resident cleanly and prepare the leasing team with the information they need for the next lease cycle.


:::info
**Scenario routing.** Pre Move-Out splits into two groups:

* **Outgoing Resident** (steps 1-6) — driven by an existing resident giving notice. **Skip this group when** `**Leasing Type = New Property**` (onboarding a vacant unit where there is no outgoing resident).
* **Leasing Team Prep** (steps 7-11) — always applies.

:::

## Objectives


:::tip

1. Set up the entire Leasing 2.0 project for success.
2. Communicate with the outgoing resident, inform the property owner, and gather the information needed for a seamless leasing experience for all stakeholders (residents, owners, vendors, staff).

:::
