Sagareus regularly receives mail and electronic notices from state and federal agencies that require a structured response — child-support income withholding orders, unemployment claim notices, and Washington ESD wage verification audits.

Each of these has a statutory response deadline printed on the notice. Missing the deadline exposes Sagareus to penalties or default judgments.


:::warning
Log every state/federal notice the day it arrives: employee name, date received, issuing agency, response deadline. Then route to the corresponding child SOP and respond by the deadline.

:::

The child docs below contain the step-by-step response procedure for each notice type — what to verify, who to notify, what to enter into ADP, what to return to the agency, and how to file the records.

## Children

* Responding to Child Support Orders — receipt, initial review, employee notification, payroll setup in ADP, Employer Response Form, ongoing administration, recordkeeping (CCPA limits, WA-vs-other-state procedural rules)
* Responding to Unemployment Claim Notices — review, factual response, deadline tracking, supporting documentation
* Responding to WA ESD Wage Verification (Fraud Prevention Audit) — verification procedure, wage report, response form
