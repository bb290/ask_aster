Customer Service scenarios for Resident Relations questions and complaints, noise, mail, property modifications, guest stays, insurance, fee waivers, and more.

## Complaints

* [CS // Stolen Mail](../../sops/resident relations/cs-stolen-mail.md): File police report, recommend secure delivery
* [CS // Stone Court / Woodlawn Package Delivery](../../sops/resident relations/cs-stone-court-woodlawn-package-delivery.md): No interior access for drivers
* [CS // Reimbursement for Damaged / Stolen Property](../../sops/resident relations/cs-reimbursement-for-damaged-stolen-property.md): Management not responsible; use renter's insurance
* [CS // Noise Complaint / Barking Dog](../../sops/resident relations/cs-noise-complaint-barking-dog.md): Collect details, issue informal notice
* [CS // Smoking Complaint](../../sops/resident relations/cs-smoking-complaint.md): Collect details, issue formal notice
* [CS // Dog Poop Complaint](../../sops/resident relations/cs-dog-poop-complaint.md): Community notice to all Residents
* [CS // Lease Status of Another Resident](../../sops/resident relations/cs-lease-status-of-another-resident.md): Privacy policy; do not disclose

## Property & Unit Questions

* [CS // Missing Internet / Coaxial Installation](../../sops/resident relations/cs-missing-internet-coaxial-installation.md): Authorization letter + up to $250 reimbursement
* [CS // Resident Locked Out](../../sops/resident relations/cs-resident-locked-out.md): No lockout service; contact locksmith
* [CS // Work in Exchange for Rent Reduction](../../sops/resident relations/cs-work-in-exchange-for-rent-reduction.md): Not allowed; exceptions for licensed pros
* [CS // TV Mount Request](../../sops/resident relations/cs-tv-mount-request.md): Professional installation required
* [CS // Artwork / Shelves / Mirrors](../../sops/resident relations/cs-artwork-shelves-mirrors.md): Lightweight OK; heavy items may incur charges
* [CS // Repaint Unit, Multifamily](../../sops/resident relations/cs-repaint-unit-multifamily.md): Allowed; restore to original color
* [CS // Repaint Unit, Single Family](../../sops/resident relations/cs-repaint-unit-single-family.md): Owner approval required
* [CS // Significant Remodel Request](../../sops/resident relations/cs-significant-remodel-request.md): Not permitted
* [CS // Landscaping Modification Request](../../sops/resident relations/cs-landscaping-modification-request.md): Owner approval required
* [CS // Window A/C Installation](../../sops/resident relations/cs-window-a-c-installation.md): Not allowed; recommend portable
* [CS // A/C Installation Request](../../sops/resident relations/cs-a-c-installation-request.md): Owner not installing; recommend portable
* [CS // Home Business Request](../../sops/resident relations/cs-home-business-request.md): Office OK; high-impact needs approval
* [CS // Inspection Timing Request](../../sops/resident relations/cs-inspection-timing-request.md): Loop in inspector; accommodation not guaranteed
* [CS // Owner, Are Window Coverings Required?](../../sops/resident relations/cs-owner-are-window-coverings-required?.md): Not legally required; recommend leaving out unless owner requests
