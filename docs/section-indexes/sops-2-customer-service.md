### CS Team | Email / Text Response

* CS team routes incoming requests — [CS Cheat Sheet](https://docs.google.com/spreadsheets/d/1CvitM8Yk2IsvHn8DHqxUpjfC2KV1b5db3G6XllqBDm0/edit?usp=sharing)


:::warning
Every incoming request requires **a response** and **an Asana task**. Most require at least 1 follow up.

:::

#### Response

* Use templated response if one is available
* Decide if additional response is needed from the team:
  * IF YES — add the team member to the thread
  * IF NO — simply respond to the sender

#### Task

* Check if existing task already exists → IF YES, update existing task
* IF no existing task → Use templated task if available (e.g., Leasing 2.0, Finance & Office, etc.)
* IF no templated task → Forward email to [x@mail.asana.com](mailto:x@mail.asana.com), then: make task public, normalize task name with address, remove email signature, assign to team member, set due date, mention staff in comment

#### Follow Up

* Create subtask "Follow Up" assigned to Customer Success Team
* Check SLA: High priority = same day, Standard = 2 business days
* If still being worked on, provide status update to original email thread

### Asana Task Creation Defaults

#### Task Title

* Format: "Specific Action or Issue // Property Address"
* Address: abbreviated, excluding State & Zip (Avenue → Ave, Place → Pl, South → S)
* Good: "Multiple Repairs Needed // 925 28th Ave S #A, Seattle"

#### Task Assignment Cheat Sheet

* Missed Turn Over work → [Turn Over Coordinator Lead](../role-profiles/turn-over-coordinator-lead.md) (Mailbox Rekey → [Turn Over Coordinator Lead](../role-profiles/turn-over-coordinator-lead.md))
* Maintenance Issue → [Maintenance Coordination Lead](../role-profiles/maintenance-coordination-lead.md)
* Misc Complaints → [Customer Service Specialist](../role-profiles/customer-service-specialist.md) (Noise, dogs, neighbors)
* When in Doubt → [Admin Team Lead](../role-profiles/admin-team-lead.md)

#### Description

* Add full description of the issues
* Add pictures when useful
* Copy / paste or upload screenshots of relevant communications
