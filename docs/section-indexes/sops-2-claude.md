**Claude** is the agentic AI we use at Sagareus. This space has everything you need to start using it.

## Common terms

* **Claude** — Anthropic's family of AI models. Comparable to OpenAI's GPT or xAI's Grok.
* **Opus** — The specific Claude model we use. Anthropic's most capable model.
* **Claude Code** — The interface (a terminal app on your computer) where Claude reads your files and uses tools on your behalf. Think of it as the "browser" through which you talk to Claude.
* **Skill** — A repeatable workflow you can trigger by typing `/skill-name`. Like a saved playbook Claude follows.
* **MCP** — A connector that lets Claude reach into another tool (Asana, Outline, Gmail, etc.) to read and write data on your behalf.

## How is Claude Code different from chatting with GPT / Grok / Claude in a browser?

Browser chat is one-shot: you ask, it answers. Claude Code is **agentic** — it can take actions:

* **Tool use** — Claude can call Asana, Outline, Gmail, Calendar, Google Drive, and more on your behalf.
* **Local file access** — Claude reads, edits, and creates files on your computer.

**Example:** Claude can pull data from Asana and Outline, reason about it following your instructions, then create tasks, update docs, draft emails, and book calendar slots — all in one conversation.

## Security

Claude Code can read and write files on your computer. **Use it on your work machine only.** Do not install it on a personal computer that holds private information you don't want exposed.
