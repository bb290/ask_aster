# Vendor // Pay Vendors

Sagareus prioritizes **EFT (Electronic Funds Transfer)** for all vendor payments. EFT is faster, trackable, and reduces manual processing.

**Payment methods in order of preference:**


1. **EFT** — Preferred. Batched in Buildium. Vendor receives confirmation email.
2. **Direct Deposit** — For vendors set up with direct deposit in Buildium.
3. **PayPal** — When vendor does not accept EFT or check.
4. **Paper Check** — Last resort for vendors without electronic payment options.
5. **Credit Card** — Only for temporary accounts or vendors not yet set up for other methods.

Standard payment terms are **Net 30**. See sub-documents for step-by-step procedures for each method.
