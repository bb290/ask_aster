## Reconciling Security Deposit Holding Discrepancies


:::info
Compare the "Security Deposit Holding" amount in the accounting system with the actual funds collected and held in the bank account. This information can be found by producing a Balance Sheet Report using the owner or property address.

:::

* **Identify Discrepancy:**
  * Compare the "Security Deposit Holding" amount in the accounting system with the actual funds collected and held in the bank account.
    * This information can be found by producing a Balance Sheet Report using the owner or property address
  * If the difference is due to a shortage then a deposit holding process needs to be performed.
  * ![image](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdfdTia0Sgy_GSBhOoIf0dUPDgTSTN_nwMY0m0eb6GxBS9KUpsAZfK0hBe4Qw3LXbr6oVPB7u8Vgxj2thUWXB9LzSEv1TtogSkdLfwOLXfVlxeBCpxBW7o12FVharsV6LffDbaF6a6lpwovScKm8SUy-hx6?key=nnhKEmcP1_mvGVmGiinqRQ)
* **Correct the Discrepancy:**
  * Verify the Total Liability amount is correct
  * Confirm there are enough funds in the owner's operating account to cover the transfer
  * Go to the owner's banking page in Buildium
  * Select "Record Other Transaction"
    * Transfer
    * Property: Property address where the funds are missing from
    * Transfer From: Owner's operating account
    * Transfer To: 9990 Deposit Account
    * Transfer Amount: Amount the needs to be transferred
    * Memo: DEPOSIT HOLDING
  * Initiate transfer in the actual bank accounts (keep in mind deposit funds are held in Chase; BofA accounts needs to go through the BXN account)
    * Chase → Chase
    * BofA → BXN → Chase


:::warning
Deposit funds are held in Chase. BofA accounts must go through the BXN account for transfers.

:::

* **Verify Reconciliation:**
  * After making corrections, re-check the "Security Deposit Holding" amount against the actual funds to ensure they now match.

## Transferring Security Deposit and/or Transferring Last Month Rent from 1 unit to another


:::tip
Three-step process: apply last month rent, refund from existing ledger, receive on new ledger.

:::


1. Apply last month rent to existing ledger
2. Refund security deposit & last month rent to Resident on existing ledger
3. Receive funds on new ledger
