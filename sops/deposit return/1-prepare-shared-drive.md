1. Create a google drive subfolder under Deposit Withholding Shared drive
   * Name: Deposit Statement // <Address> // <MO - YEAR>
   * Update sharing setting to be viewable for everyone
     * CTRL+ALT+A
     * Right click folder → Share → Share

 ![](/api/attachments.redirect?id=3377cd87-f9a7-4061-aa5f-fe43bce2b702 " =510x185")


:::warning
Each address subfolder is shared with "Anyone with the link" can "view" only — This protects Resident name / address from being shared with other Residents

:::


2. Populate the drive with the following files and rename to match:
   * Deposit Refund Letter
   * Move-in Inspection
     * Use report and pull the one with signature
   * Move-out inspection
   * **Deposit Withholding Details**
     * Create a Google Sheet using the template "[Deposit Withholding Details](https://docs.google.com/spreadsheets/d/1haPx9iqAS83bWiVFULIb7QsqHUGLrrEecSapiav8X3I/edit?usp=drive_link)"
   * Attach invoices:
     * Invoices - Turn Over
     * Invoices - Repairs
     * Invoices - Cleaning
     * Search all bills from the property for the last 60 days in BD
3. Add the link of the shared drive to the Process Deposit Asana task
