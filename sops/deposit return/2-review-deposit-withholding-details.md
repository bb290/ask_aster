### Review Move-out inspection


1. Review the move-out inspection and identify items needing withholding from comments and pictures. Follow [Deposit Withholding Requirements](deposit-withholding-requirements.md)
2. Use Adobe PDF and highlight the line items and pictures - Save the file

### Sanity check Move-in inspection


1. For each highlighted line item, check the Move-in inspection to see if the line item is really caused by the Resident.
2. If not supported by the move-in inspection, remove the highlight from the move-out inspection

### Review Invoices & Add to Deposit Withholding Details


1. For each line item on the move-out inspection report, check if there is a corresponding line item in the vendor invoice
   * **IF YES** - add the line entry to the Deposit Withholding Details spreadsheet. Snip and copy & paste supporting pictures
   * **IF NO** - add a line entry using the [Common Charges](deposit-withholding-requirements.md)
