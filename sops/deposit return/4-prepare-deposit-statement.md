1. Download [Deposit Refund Letter](https://drive.google.com/file/d/1Hboxmos5O4dm6w99_kYOOfTb82kWVyhH/view?usp=drive_link)
   * Add the shared google drive link
2. Download [Deposit Refund Statement](https://drive.google.com/file/d/19Jb75BgpaDNHhhvXHQkgZR-pToIkbeeq/view?usp=drive_link)
   * Fill out the form completely
   * Double check to make sure the number matches the ledger
3. Merge the final package, rename it as **Deposit Refund Package - <Address>**. The package should include the following:
   * Deposit Refund Letter
   * Deposit Withholding Details
     * Export to PDF from the spreadsheet
   * Deposit Refund Statement
   * Full Resident Ledger
   * Backing Invoices
4. Upload the final package to the Google Shared Drive
