When a vendor reports the key box is not found at the property:


1. **Check if a resident has already moved in.** Audit the LU task in the parent leasing task and confirm if the resident has already moved in.
   * **IF the resident has moved in** -- follow the [Property leased out prior to turn over completion](turn-over-property-leased-out-prior-to-turn-over-completion.md) protocol
2. **If no resident has moved in**, communicate with the leasing agent to confirm access instructions. Check the LU or Move-in task to identify the leasing agent.
3. **Do NOT reach out to the owner** unless this is an owner managed turn over.
