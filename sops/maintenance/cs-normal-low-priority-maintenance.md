> **Situation:** Resident reports a maintenance issue that does not involve immediate safety, habitability, or major property damage risk.


---

## Key Points

* **Standard workflow** — Assign to Maintenance Coordinator with today's due date
* **Photos help** — Request photos if not already provided to expedite resolution
* **Buildium link** — If request came through Buildium, add the Task link and Resident's message in the description


---

## Action


1. Move to Maintenance Project; New Section
2. Update Fields
   * PRIORITY Field to Normal
   * Ticket Submitted Date: Today
3. Assign to Maintenance Coordinator (Joy)
4. Due: Today
5. In the Description area, add the Task link and Resident's message if the request was received in Buildium. If the request was taken via phone call or email, add the BD Unit Link and write a brief description of the request.
