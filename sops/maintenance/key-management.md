## Key Management

Sagareus is responsible for maintaining up-to-date keys for all managed properties to ensure prompt access in the following situations:

* Emergency access to property
* Vendor access for repairs or maintenance
* Access for evictions
* Access for inspections

### Key Storage Locations


:::info
* **Bellevue Office**
  * Keys are stored in two secured key vaults.
* **On-Site Storage**
  * For properties with more than four units and a secure location, a keyvault may be placed on-site.

:::

### Key Tracking & Security


:::warning
* **Tracking System:** Key locations and access information are maintained in 1Password using the "Property Access" vault.
* **Access Control:**
  * Access to the vault is strictly controlled and granted on a need-to-know basis.
* No property access info should be listed in Asana or Email

:::

## Key Entry


:::info
* The title of the entry is the property address // CabinetSlot
  * Ex for cabinet A, slot #28: 315 Pelly Ave N, Renton // A28

:::


---

## Onboarding New Property


:::info
* During property onboarding, document all keys received.
* Add a new entry for the property in the "Property Access" vault in 1Password.

:::


---

## Move-Out


:::info
* The leasing agent will install a keybox on the property and place the returned key in the keybox

:::


---

## Turn Over


:::warning
* Sagareus' policy is to rekey the doors between each tenancy

:::


---

## Annual Inspection


:::tip
Annual inspection is our opportunity to verify the keys we have on-hand.

:::


---

## Resident Report Loss of Key (Never Provided)

* Post move-in Resident may report some keys were not provided
* Dispatch to Approved Locksmith
* Update billing to track the expense as Sagareus expense

## Resident Report Loss of Key (Lost by Resident)

* Provide Approved Locksmith contact to the Resident and ask them to schedule directly with the locksmith
* Resident responsible for paying locksmith directly
