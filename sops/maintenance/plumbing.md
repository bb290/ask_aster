# Kitchen/Sink/Shower/Toilet Backup

## Dispatch


:::warning
Dispatch is time-sensitive. Backup issues must have a vendor confirmed within 24 hours, or escalate immediately.

:::


1. Dispatch to Approved Rooter
2. Get confirmation if vendor can be there within 24 hours

## Triage


:::warning
Triage does NOT block dispatch. Collect information in parallel to dispatching vendors.

:::


1. IF backup is continuous: **This is an active emergency!!**
2. IF water damage exists: Ask if the water damage is caused by sewage or water?

## Hydrojet Request


:::warning
**Approval thresholds for hydrojet requests:**

* IF **drain is draining** and the vendor is just recommending it, **REJECT** the work.
* IF drain is still clogged, approve the hydrojet if it is under **$750**.

:::

## Sewer Scope/Camera Request

Sewer scopes are critical for diagnosing larger sewer line issues.
