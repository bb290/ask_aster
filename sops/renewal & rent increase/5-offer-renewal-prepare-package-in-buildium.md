## Offer Renewal in BD


1. If owner approves the renewal, offer renewal in BD
2. Navigate to [BD Lease Renewals](https://sagareus.managebuilding.com/manager/app/leasing/lease-management/renewals/not-started)
3. Click on Renew Lease for the lease

 ![BD Lease Renewals screenshot](/api/attachments.redirect?id=c5f52cf1-d29b-46ec-a812-bcee65f5c666)


4. Enter Lease details:
   * Lease Type: Fixed with Rollover
   * Add all the rent charges and the associated type
     * Rent
     * Utilities
     * Pet Rent
     * Parking Rent
     * Others
5. Select **Renew and Prepare E-Signature**
6. Use the **Renewal** template


:::warning
**IMPORTANT: Append the Local jurisdiction disclosure to the end of the renewal package**

[Local Disclosures](https://drive.google.com/drive/folders/19HW7Y7GcFftnTKdOjAEqvednOnTsRv3G?usp=sharing)

:::

## Preparing the Renewal Package


1. Once the Renewal is sent via e-signature
2. Download the PDF locally and rename to "Renewal - [Address].pdf"
3. Attach the file to the Renewal task and Serve document task
