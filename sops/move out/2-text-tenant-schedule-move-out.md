**When to Use This Page**

> *Use this page if:*
>
> * *you've been assigned the* `*Text | Tenant - Schedule Move Out*` *subtask*
> * *you need the canonical scheduling text template for the outgoing resident*

**Situation Summary**

Roughly 14 days before the move-out date, text the outgoing resident to confirm the keybox install window, share the tenant lockbox code, and set expectations for the inspection. This gives the resident enough time to plan key handoff and move-out prep.


---

## Procedure


1. Calendar block 2–3 hours for the keybox install and the move-out inspection (adjust for property size).
2. Find the resident's contact info on the Move Out subtask description. If missing, check the parent Leasing task or look up in Buildium.
3. Send this text template to the resident (fill in bracketed fields):

> *Hi \[Resident Name\], I've been assigned to your move-out. I'll be installing a key box at the property on \[date\] between \[time\]. The code is 2001—please leave the keys inside. Don't forget the mailbox key & garage door openers 😁 Your move-out inspection is scheduled for \[date\]. Let me know if you need additional time. For move-out procedures and FAQs, please check this link:* [*Move-Out Procedures*](../../docs/section-indexes/sops-2-move-out.md)*. Let me know if you have any questions! Best, \[Your Name\]*


4. Mark this subtask complete once the text is sent.
