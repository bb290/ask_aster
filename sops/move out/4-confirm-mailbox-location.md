**When to Use This Page**

> *Use this page if:*
>
> * *you've been assigned the* `*Confirm | Mailbox Location*` *subtask*
> * *you're already on-site installing the keybox and want to confirm mailbox location in the same trip*
> * *the Mailbox field on the property is listed as "Unknown"*

**Situation Summary**

The Mailbox field on Property Settings drives Lease Up handoff — the incoming resident needs a correct mailbox location to receive mail. Confirming this during the keybox install trip avoids a second site visit.


---

## Procedure


1. While on-site for the keybox install, compare the mailbox location in person to what is noted on the Move Out task (and Property Settings Mailbox field).
2. If the listed location is correct → mark the subtask complete, no further action needed.
3. If the listed location is wrong → update the Mailbox field on Property Settings to the confirmed location, then mark the subtask complete.
4. If the field is listed as "Unknown" and you cannot locate the mailbox yourself, contact the resident with the template below.


---

## Contact-Resident Text Template

> *Hi \[Resident Name\], I was on-site installing the keybox today and attempted to locate the mailbox for your unit but couldn't find it. Can you please confirm the location and mailbox number? Thank you!*

Once the resident responds, update the Mailbox field on Property Settings and mark the subtask complete.
