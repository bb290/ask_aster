**When to Use This Page**

> *Use this page if:*
>
> * *you've been assigned the* `*Change | Lockbox code & Update 1Password*` *subtask*
> * *the resident has vacated and the inspection is complete*

**Situation Summary**

On move-out day, rotate the lockbox from the tenant code (2001) to the internal code. This removes resident access while keeping staff and vendors able to get in for Turn Over work. Every rotation must be reflected in 1Password so the full team has current access.


---

## Procedure


1. Open 1Password and look up the canonical internal lockbox code for Sagareus-managed properties.
2. At the property (or remote, if the box supports remote updates) change the lockbox from the tenant code (2001) to the internal code.
3. Test the new code physically before you leave — confirm it opens the box.
4. Update the 1Password entry for this property with the new code and the change date.
5. Mark this subtask complete.


---

## Key Rules (short form)

* **Never share internal codes with tenants.** These codes provide access to all listed properties and must stay internal.
* If you're unsure which internal code to use, stop and ask — do not guess or leave the box on a non-canonical code.

For the canonical list of lockbox codes and when to use each, see [Key Management](../maintenance/key-management.md).
