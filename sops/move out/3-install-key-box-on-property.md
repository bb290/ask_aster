**When to Use This Page**

> *Use this page if:*
>
> * *you've been assigned the* `*Install | Key box on property*` *subtask*
> * *you need to install a lockbox so the outgoing resident can leave keys without a scheduled handoff*

**Situation Summary**

Installing the lockbox up to 10 days before the move-out date lets the resident leave keys at their own pace and lets staff access the unit without coordination. The lockbox starts on the tenant code (2001) and is rotated to the internal code in step 7 after move-out.


---

## Procedure


1. Install the lockbox on the property. It can be installed up to 10 days before the move-out date.
2. Set the lockbox code to the tenant code (see [Key Management](../maintenance/key-management.md) for the canonical code — currently 2001).
3. Text the resident before you arrive to let them know you're coming, and text again once the lockbox is installed. Include the lockbox location and the code.
4. Mark this subtask complete.


---

## Key Rules (short form)

* **Tenant code = 2001.** If you're unsure how to change the code on the lockbox, ask for help — do not leave it on a different code.
* **Never share internal codes with tenants.** Internal codes provide access to all listed properties and must stay internal.
* **If tenant doesn't leave the key:** If we communicated properly and they still didn't leave the key, it comes out of their security deposit. If we failed to communicate and they couldn't find the lockbox, that's a Sagareus expense. Communicate.

For the canonical list of lockbox codes and when to use each, see [Key Management](../maintenance/key-management.md).
