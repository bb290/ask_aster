**When to Use This Page**

> *Use this page if:*
>
> * *you've been assigned the* `*Confirm | Mailbox Key Available & Works*` *subtask*
> * *you're on-site for the move-out inspection and want to confirm the mailbox key in the same trip*

**Situation Summary**

The incoming resident will need mailbox access from day one of their lease. Confirming the mailbox key now — while the unit is vacant and still in our active scope — avoids emergency locksmith dispatches during Lease Up. When a key is missing, the Turn Over Coordinator (Bart) handles the locksmith so the cost can be charged against the outgoing resident's security deposit.


---

## Procedure


1. Locate the mailbox key the outgoing resident left (usually in the keybox or inside the unit).
2. Walk to the mailbox and physically test the key — confirm it opens the mailbox.
3. Handle the outcome:
   * **Present and working** → mark the subtask complete.
   * **Missing or not working** → follow the "Missing Key" section below.


---

## Missing Key

Do **not** dispatch the locksmith yourself. The Turn Over Coordinator handles this so the cost can be deducted from the outgoing resident's security deposit.


1. Confirm a mailbox key is actually needed for this property (some units have cluster-box keys assigned by USPS; in that case, note it and complete the subtask).
2. Add a comment to the subtask: *"No mailbox key at move-out."*
3. Reassign the subtask to the Turn Over Coordinator (Bart).
4. Do not mark the subtask complete — Bart will close it after the locksmith has rekeyed the box and the cost is logged against the deposit.
