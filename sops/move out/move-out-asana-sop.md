**Asana Project:** [Leasing 2.0](https://app.asana.com/) -- Move Out subtasks on the parent `Leasing | <address>` task


:::info
Follow these steps in order from top to bottom -- they map to the subtasks in the Asana Move Out template.

:::


1. `1 — Update | move_out_date metric` -- Record move-out date on parent Leasing task; drives downstream scheduling and KPIs
2. `2 — Text | Tenant - Schedule Move Out` -- Notify outgoing resident of keybox install date, code, and inspection schedule
3. `3 — Install | Key box on property` -- Install lockbox with tenant code 2001; see Key Management for code rules
4. `4 — Confirm | Mailbox Location` -- Verify mailbox location on-site; contact resident if listed as Unknown
5. `5 — Conduct | Move-out inspection & share report` -- Complete Move-Out Inspection in the HappyCo Inspector App and email to leasing@sagareus.com
6. `6 — Record | Move-out video` -- Record and upload a Move-Out Video walkthrough for documentation and marketing
7. `7 — Change | Lockbox code & Update 1Password` -- Rotate from tenant code to internal code; update 1Password
8. `8 — Confirm | Mailbox Key Available & Works` -- Test the mailbox key; reassign to Turn Over Coordinator if missing
