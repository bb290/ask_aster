## Roommate Change or Request for Subletting


1. Leasing Assistant receives task from customer service, or
2. Leasing Assistant receives Request directly from Resident
   1. Utilize [Customer Service Response](../../docs/section-indexes/sops-2-customer-service.md)
   2. Create Asana Task
      * Project: [Leasing 2.0](https://app.asana.com/1/706990140225747/project/1208297375044026/list/1208297386032994)
      * TASK TEMPLATE: "Roommate Addendum // <ADDRESS>"
      * Section: Pending Applications
      * Assigned to: Leasing Assistant (Mary or Jody)
      * Due: 2 days
      * Screenshot communication, upload to comments

## Roommate Addendum Process


:::info
Follow these steps sequentially once the roommate change request has been received and the Asana task is created.

:::


1. Note Details in Description
   1. Exiting Resident
   2. Adding Resident
2. Update task as details come in
3. Follow up every 2 days until Application is received:
   1. Follow same procedures as Initial Applicant Screening:
      1. Charge Application Fee
      2. Request Proof of Income
      3. Review Application for completeness
      4. Proof of income (verify it meets income standards)
   2. Assign to Leasing Manager for Final Approval
      1. If approved, proceed.
      2. If denied, send Adverse Action
4. Prepare & Send Addendum for Signature
   1. PandaDoc → Templates → Lease + Move In Instructions folder → Roommate Addendum
   2. Send to all existing and new Residents for e-signature via PandaDoc.
   3. Once completed, attach the signed addendum to the Lease Record in Buildium.
5. Apply Change in Occupancy Fee (EXCEPT in SEATTLE)
   1. Add $200 Change in Occupancy Fee to the existing Resident's ledger.
      1. Category = Admin CC


:::warning
Do NOT apply the $200 Change in Occupancy Fee for Seattle properties. Seattle regulations prohibit this charge.

:::


1. Update Resident Account
   1. Add the new Resident to the account in Buildium.
   2. Confirm the new roommate has access to the resident portal (if applicable).
2. Close Task
   1. Confirm all steps are completed and documentation is filed.
   2. Mark the Asana task as "Complete."

## Subletting


:::info
Subletting follows the same process as a roommate change. The original Resident(s) are treated as the "exiting Resident(s)", and the incoming sub-tenant must complete the application and approval process outlined above.

:::
