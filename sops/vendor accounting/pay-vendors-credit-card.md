Credit card payments are used only for temporary accounts or vendors not yet set up for other payment methods.

## When to Use

* Bill can be paid via CC but vendor is not yet set up for EFT/check/direct deposit
* Temporary vendor account before full onboarding is complete

## Procedure

* If a bill is able to be paid via CC, make payment and issue a temp loan to account to reimburse CC
* If there is only Direct Deposit or Check options available, push out until banking account is set up
