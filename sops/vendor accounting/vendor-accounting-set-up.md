## Asana Setup


1. Check if Vendor exists in the [Active Vendor List](https://app.asana.com/1/706990140225747/project/1210537535377048/list/1210537537583349)
   * **If no → Create Asana Task**
2. Project: Vendor Relations 2.0
3. Task Template: \[Company Name // Vendor Contact Name\]
4. Subtask > Accounting Set Up: Assign to Ana
5. Due: Today / Next Day
6. Upload Invoice or vendor contact information (email is required)

## Adding a New Vendor to Buildium


1. **Navigate:** Maintenance > Vendors > Add Vendor
   * A new vendor can also be added under "Record Bill" > "Pay to" > "+Add vendor"
2. **Name / Company Name**
   * Enter the full name of the vendor if not using a company name or LLC
   * Enter the company or LLC name the vendor operates under
   * If using a company name, select the "company" box after entering the name
   * If vendor is set up for Direct Deposit, add **EFT** after their last name or company name
3. **Primary Email**
   * Enter the main contact or accountant email address
   * Emails are sent for each EFT payment — must be entered before payment goes out
4. **Phone Number**
5. **Address**
   * Mailing address is required for paper check payments and 1099s
6. **1099 / W9**
   * Upload W9 to vendor profile under "files"
   * Select if paid under SSN or EIN
   * Add SSN or EIN under "Taxpayer ID"
7. **Insurance Provider**
   * Upload Certificate of Insurance to vendor profile under "files"
   * Input insurance info: provider, policy number, expiration date
8. **Business License**
   * Upload business license to vendor profile
