1. Create a new Google File under `Rental Listing Marketing`, labelled with the property address.
2. Copy the share link.
3. Paste the link under two subtasks:

   
   1. Move-Out → Move Out Video
   2. LU → Listing Video
4. Allows the Leasing Agent to quickly upload videos to the correct folder while in the field.
