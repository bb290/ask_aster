:::info
Step-by-step guide for all time-off scenarios. For policy details and accrual rates, see [Work Schedules + Out of Office](work-schedules-out-of-office.md).

:::

## Partial Day (1–7 hours)

No approval required.


1. Add event to **Team Calendar** → \[Name // OOF\] with date and time
2. Submit PTO in **ADP**

## Planned Day(s) Off (1+ days)

Manager approval required — reviewed every **Thursday 12:30–1:30pm**.


1. Check **Team Calendar** for conflicts
2. Add event → \[(PENDING) Name // OOF\] as all-day event
3. Submit Holiday or PTO request in **ADP**
4. Wait for ADP notification:
   * **Approved** → Manager updates calendar to \[Name // OOF\]
   * **Declined** → Manager removes event


:::tip
Request early and coordinate around existing team absences for best chances of approval.

:::

## Calling in Sick / Unplanned


1. **Email or text** your manager/team lead and [HR](../../docs/role-profiles/hr-manager.md) before your workday starts
2. Submit PTO in **ADP** when you're able
3. Manager handles your Team Calendar and task reassignment

## Extended Leave (PFML)

Apply directly through the state — not through Sagareus.


1. Review eligibility at [paidleave.wa.gov](https://paidleave.wa.gov/)
2. Give your employer **30 days written notice** when possible ([notice form](https://paidleave.wa.gov/app/uploads/2019/12/30-day-notice-to-employer.pdf))
3. [Apply through WA Employment Security Department](https://paidleave.wa.gov/get-ready-to-apply/)
