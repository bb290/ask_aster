## How Does Banking Work?

Each bank account is set up for EFT processing through the Buildium epay application process. To apply for EFT follow these steps.


 1. Select **Accounting** in Buildium Sidebar.
 2. Click **Add Bank Account**.
 3. Create bank account name to include 3 digit unique numerical code. Typically new accounts are set up in the 900s while they are being onboarded.
 4. Enter routing and account number of the actual bank account.
 5. Click **Create account**. The bank account will now be created. Navigate to menu item **Payment Settings**.
 6. Click **Apply for ePay Account**.
 7. Select **Property Management Company**.
 8. Upload required documents. I typically print off the checking statement and screenshot of routing and account number. Upload a combined PDF of this document.
 9. Select **Incoming and outgoing transactions**. Select 2 units.
10. Continue with application. Use average rent of $3000 for two units to multiply and calculate the expected transactions limits necessary to complete the application. Select **$8000** as max EFT accepted. This prevents further paperwork required which we wont have because we typically onboard brand new banking accounts without transaction histories.
