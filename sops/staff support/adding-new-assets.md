Once new asset is received:


1. Use label maker and create a **unique Item ID** (verify current item IDs in the [Asset Check Out](https://docs.google.com/spreadsheets/d/1n2k7oY9oGUadTmFZWtDe7DN47BAhI0L9YnF-Mri5K1U/edit?usp=sharing) spreadsheet first)
2. Each item must start with a letter prefix followed by a sequential 3-digit number starting at 100 (e.g., H100, H101, F100):
   * Heater: H###
   * Fridge: F###
   * AC Unit: AC###
   * Dehumidifier: DH###
   * Microwave: M###
   * Fan: Fan###
   * Ozone Generator: OZ###
3. Double check asset log for current ID numbers to avoid duplicates
4. Take a picture of the asset and save it to [Asset Photos](https://drive.google.com/drive/folders/1JP_Bve2NE3Ch-7H-sOifGHjd0eXgip_Y?usp=drive_link)
   * File name is the Item ID (e.g., H105)
5. Add new entry to [Sagareus Asset Check Out](https://docs.google.com/spreadsheets/d/1n2k7oY9oGUadTmFZWtDe7DN47BAhI0L9YnF-Mri5K1U/edit?usp=sharing&gid=198306450):
   * Item ID
   * Type
   * Location
   * Status
   * Notes/Task
   * Photo Link
