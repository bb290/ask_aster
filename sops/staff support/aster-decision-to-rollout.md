Manager skill: turns a reviewed staff capture (incident or edge case) into Asana rollout work with itemized subtasks.

## When to Use

A New captures task has been reviewed and the next step is "update SOPs + train the team." Invoke /decision-to-rollout or say "roll out this capture." Not for unreviewed or informational-only captures.

## How It Works

1. Paste the capture task link; Aster asks for the decision and which SOPs to update.
2. Aster generates proposed SOP modifications using current SOP content from its corpus.
3. Creates a parent task plus subtasks in Roll Out, Ready for Roll Out, with proposed changes as ready-to-paste markdown in each subtask.

:::warning
This skill does NOT modify SOPs. It creates the work items; a human finalizes the actual SOP edits.
:::

Requires Asana and manager access to the Roll Out project.
