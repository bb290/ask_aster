## RingCentral Set Up

See also: [RingCentral Set Up (Google Doc)](https://docs.google.com/document/d/1uNqdXTs8de7M6VyV_KtJ2xsfEGX2LDs45W9ZzQAdetM/edit).

### Turn Off Email Notifications

* User List
* Click User
* Notifications tab
* Messaging Notifications
* Uncheck all notifications

### Enable Text Messaging

* From Admin Portal
* Phone System → Business SMS → Set Up SMS
* Click Add Numbers Button
* Choose new number
