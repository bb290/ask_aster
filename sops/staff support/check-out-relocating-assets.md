:::warning
All asset checkouts must go through [Assistant Property Manager (Asset)](../../docs/role-profiles/assistant-property-manager-asset.md).

:::

## Procedure

[**Assistant Property Manager (Asset)**](../../docs/role-profiles/assistant-property-manager-asset.md)**:**


1. Identify asset to check out
2. Provide instructions on how to access the asset
3. Create a templated task in [Maintenance 2.0](https://app.asana.com/1/706990140225747/project/1210320631715650/list/1210322778568813) project using "Asset Retrieval — <Asset> // <Address>"
   * Include link to the related task that requires the asset
   * Include Asset ID in the task
   * Set due date to 2 weeks out to review the related task and see if the asset can be retrieved
