Ask Aster is Sagareus's internal AI teammate: it answers operational questions grounded in our SOPs and runs built-in skills for leasing work, staff captures, and management workflows.

:::info
**Getting set up:** connect the Aster MCP in Claude.ai along with the Asana, Gmail, and Google Calendar connectors, then run /meet-aster for a guided introduction.
:::

## Asking Questions

Type any operational question in plain English. Aster searches all Sagareus SOPs and answers with the source linked. If the answer is not in the SOPs, file /edge-case.

## Ground Rules

- Aster reads SOPs; it never edits them. SOP changes go through the rollout process.
- Skills that write (Asana tasks, Gmail drafts) always show the draft before anything is filed or sent. Nothing auto-sends.
- If Aster's answer contradicts an SOP, the SOP wins; flag it with /edge-case.
