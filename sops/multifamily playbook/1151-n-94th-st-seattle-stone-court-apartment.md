**Maintenance Room Access & Unit Keys**

* Locate the Maintenance Room
* The key is in the lockbox in the meter room.
* lockbox installed behind the meter room door. Code in 1P.

 ![](/api/attachments.redirect?id=91550bed-4058-4425-985c-94a6179da671)

* Access the Main Key Cabinet
  * After entering the Maintenance Room, you will find the key storage system on the back wall. This system consists of a small silver key safe and a large beige key cabinet.

 ![](/api/attachments.redirect?id=d63b3e31-f00f-4350-bebc-24994e7ac18a)

* Follow this two-step process to access the main keys:
  * Open the Small Silver Key Safe
    * Locate the small, silver-colored key safe mounted on the wall.
    * You will need the specific code or key to open it. This safe contains the key for the larger cabinet.
  * Unlock the Large Beige Key Cabinet
    * Inside the silver safe is a special key attached to a chain. Remove this key and use it to unlock the large, beige key cabinet to its right.
    * Inside the main cabinet, all keys are labeled and organized by number. Please find the specific key(s) required for your work order (e.g., unit number, elevator, storage).

 ![](/api/attachments.redirect?id=5171a449-d7f8-4bda-8c3b-48352c294c54)  ![](/api/attachments.redirect?id=3d7b0ebb-788e-4126-9a6d-031cbf9c60c0)

* Key Return Procedure
  * Before you leave the site, you must:
  * Return all keys to their designated hooks inside the large beige cabinet.
  * Lock the large cabinet using the special key.
  * Return the special key to the small silver safe and ensure it is securely locked.
  * Confirm the Maintenance Room door is closed and locked behind you.
