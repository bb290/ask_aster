## Access & Layout

### Building & Unit Access

Keys to access the building are located in a key box on the brick wall at sidewalk level to the left of the entry stairs on the 308 Bell St side. (See photo.)

* The box has a rubber weather-proof coating over the top of it.
* The code to open the box is in 1Password.
* Please note you also have to use this code to close the box.
* The key inside will access:
  * Both gates for 304 and 308
  * The back gate from alley
  * All main entry doors

 ![kix.rk42v9pe3zpc](/api/attachments.redirect?id=4939a78f-5840-4aab-ae20-d265edb95c24)

**Exterior gates:**

* Two exterior gates for 304 & 308 main entries, located on Bell St side.
* One exterior gate located in the alley, connects to the courtyard.
* 308 keys unlock 308 main entry and alley gates; 304 keys only unlock 304 main entry.
* These keys are found in the keybox on the 308 Bell St side (code in 1Password).

**Unit 33 (#A, #B, #C, #D):**

* These units are single bedroom with a shared bathroom.
* Bathroom is maintained by cleaning vendor monthly.
* The units DO NOT have a kitchen.
* Electric panels are located in #A.
  * IF ANY unit reports an electrical outage, ask #A to check the panel and reset the breaker.
