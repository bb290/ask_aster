Post the listing live in Buildium and record the list_date metric for KPI reporting.


---

### When This Runs

After [6 — Decide | Occupied showings? (y/n)](lease-up-asana-sop.md) has locked in the showing strategy and photos are uploaded. This is the formal "go live" moment for the listing.


---

### Post Listing in Buildium

Buildium → Leasing → Listings → Unlisted tab → Search property address


1. Edit Listing Details:
   * **Rent** — per agreed listing strategy
   * **Deposit** — $100 less than asking rent
   * **Availability Date** — today's date, or a future date if PreListing
   * **Contact** — assigned Leasing Agent
2. Click **"Post Listing"**


---

### Update list_date Metric

Once the listing is live, update the **list_date** field on the LU Asana parent task to today's date. This drives the days-on-market KPI.
