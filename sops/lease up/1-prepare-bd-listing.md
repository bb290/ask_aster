Gather all property details and fill in the Lease Up task so the Leasing Team has everything they need to market the property and conduct showings without interruption.

## Fill In Lease Up Task Description


1. Open the LU | [Property Address] Asana task.
2. Fill in all fields in the task description:
   * **Owner Contact** — name and preferred contact method
   * **Mailbox** — location and key/code details
   * **Parking** — number of spaces, type, and any instructions
   * **Internet Provider** — who provides service to the unit
   * **Any relevant showing notes** — gate codes, building entry, lockbox location, quirks another agent needs to know


:::warning
Another Leasing Agent may show this property. Every detail they need to conduct a showing independently must be in the task description.

:::

## Fill In Rental Policies Template

Fill in the Rental Policies template located in the 'Prep BD Listing' subtask. This is copied into every rental listing, customized per unit.

**Find property details in:**

* Previous listing
* BD (Buildium) Property Notes
* Onboarding Form (sometimes embedded in the PMA document)
* Last Annual Inspection report
* Move In Inspection report

Cover the following in the template:

* **Pet Policy** — pet friendly or not, species, weight limit, monthly fee
* **Utilities** — Accounting Team Lead completes this section;
* **Landscaping** — included in rent or Resident responsibility
* **Parking** — type, count, included or additional cost
* **Laundry** — in-unit, onsite, or none (include nearest laundromat if none)

See [Rental Listing Standards](rental-listing-standards.md) for the full Rental Policies template and listing standards.
