Trigger: Phase 4 complete, application approved. Owners: Leasing Assistants (execution) + Approved Leasing Agent or Brittany (internal review). Part of the [Tenant Placement SOP](https://sagareus.getoutline.com/doc/tenant-placement-sop-0Ho4i49zol).

TP lease requirements, non-negotiable: must use the RHAWA Lease Template, never Sagareus-branded lease documents; the lease is signed by Tenant and Property Owner only, NOT the Leasing Manager; Sagareus is not a party to the lease.

## Workflow

5.1 — Populate the TP Lease template (RHAWA base) in PandaDoc using data from Buildium, the JotForm submission, and the screening file. (Leasing Assistants)

5.2 — Internal review (Approved Leasing Agent or Brittany) before sending: correct property address and legal description; rent, term, and start date align with the application and owner's instructions; all owner-specific clauses from the JotForm incorporated; no Sagareus branding anywhere.

5.3 — Send via PandaDoc with sequenced signing: owner signs first (owner's signature = approval of lease terms, no separate approval step), tenant signs second. (Leasing Assistants)

5.4 — Confirm full execution: PandaDoc notifies Leasing Assistants and the Approved Leasing Agent automatically.

5.5 — Deliver the Welcome Home Letter (created in Phase 1) to the tenant upon full execution. (Leasing Assistants)

5.6 — Assemble the Underwriting File in Google Drive: application, AI Summary of applicant, credit report, rental history if available, income verification. (Leasing Assistants)

5.7 — Deliver the Underwriting File to the owner via the Underwriting File Delivery email, AFTER the lease is fully signed. (Leasing Assistants)

Phase complete when: both parties signed in PandaDoc, Welcome Home Letter delivered to tenant, Underwriting File delivered to owner.
