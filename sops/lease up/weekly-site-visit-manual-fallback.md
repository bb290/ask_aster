> **Situation:** The Site Visit tool (https://www.sagareus.com/field) is down, or you have no signal and the offline resubmit fails. The visit still happens on cadence and the documentation standard does not change.

Process and cadence: [Weekly Site Visit](weekly-site-visit.md).

---

## Key Points

- **The cadence never pauses** — at least one visit per week from move-out to move-in, at your Preferred Slot 1.
- **Document or it didn't happen** — one comment per visit on the **Weekly Site Visit / Inspection** subtask, with at least 2 photos.
- **The Accountability Rule still applies** — issues documented as assigned Asana subtasks with due dates, 72 hours before move-in, do not count against leasing agent performance. Manually created tickets satisfy this the same way tool tickets do.

---

## Procedure

1. Walk the property with the checklist below. Copy it into a comment on the property's **Weekly Site Visit / Inspection** subtask (create the subtask under the Lease Up task if it is missing), mark each line, and attach at least 2 photos.
2. For **every N and every open-ended task you spot**: create a ticket named `Item | Address`, put the issue detail in the description, attach that item's photos, assign it to the **Turn Over Coordinator** (occupied properties: the lead maintenance coordinator), and set a due date that respects the 72-hour Accountability Rule.
3. Write your visit notes in the same comment: **minimum 3 full sentences**. What did you observe? What did you check or test? Anything worth noting outside? See the quality documentation standard in [Weekly Site Visit](weekly-site-visit.md).
4. Move the inspection subtask's due date to your next Slot 1 day (no Slot 1 on the property: push it 6 days) so the cadence never stalls.

## Manual Checklist

```
SITE VISIT — [date]

ESSENTIALS (if these pass, someone can move in)
( ) 3 Front Keys — each tested in the door, rekeyed since last resident
( ) 2 Mailbox Keys — each tested, box empty
( ) Photo Clean — lights on, blinds open, counters clear, showing-ready
( ) Move-In Clean — no odor, scrubbed, cabinets/closets empty, zero prior resident items
( ) Smoke & CO Detectors — each bedroom + common area, CO per floor, no chirps
( ) Pests + Mold — kitchen first; sills, grout, under sink, behind washer

APPLIANCES + SYSTEMS (run every single one)
( ) Refrigerator   ( ) Oven + Stove   ( ) Microwave   ( ) Dishwasher
( ) Disposal       ( ) Washer         ( ) Dryer       ( ) Heat & A/C

MOVE-IN READY (the last mile before handoff)
( ) Quick Clean   ( ) Landscaping   ( ) Garage Remote   ( ) Plumbing & Lights

RESULT
( ) All good, photos attached
( ) Issues found, tickets created: [Item | Address, ...]

Photos attached: [#]
```

## Owner Email

The site visit itself sends no email: it feeds the weekly report. In Turnover state the weekly report **is** the site visit email; use the Turnover template in [Weekly Activity Report // Email Templates](https://sagareus.getoutline.com/doc/weekly-activity-report-email-templates-Nbjth1yCLc) with your observations, subtasks created, and photos.
