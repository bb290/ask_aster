Trigger: Phase 3 complete, listing live. Owner: Nicole (Leasing Agent). Part of the [Tenant Placement SOP](https://sagareus.getoutline.com/doc/tenant-placement-sop-0Ho4i49zol). Showing and application procedures inherit from the standard Lease Up SOP (Conduct Showings, Speed to Lead, Follow Up & Secure Application, Lease Up Policies). TP-specific rules:

## Showing Limit

* Each TP engagement includes 25 showings.
* Additional showings are billed to the owner at $50 per showing.
* Overages are batched and invoiced at close-out (Phase 7).

## Showing Documentation

Every showing must be logged; overages generate billable charges, so accurate documentation is mandatory. Showings are tracked via the Showings Log (separate task list per property in Asana). Until the Showings Log structure is built, the Leasing Agent manually tracks the showing count on the master TP task in a dedicated field or comment thread.

## Showing Count Reporting

The owner receives the running showing count in the [Weekly Activity Report](https://sagareus.getoutline.com/doc/email-owner-weekly-activity-report-VDE6GnaYef).

## Approaching the 25-Showing Limit

At showing #20, the Leasing Agent notifies the owner the property is approaching the included-showings limit.

## Applications

* Processed using the TP Application Task Template in Asana (not the standard LU template).
* Screening procedures (credit, income verification, rental history, AI Summary) are identical to full PM standards.

Phase complete when: application approved, applicant confirms intent, file ready for lease preparation.
