## When This Runs

New Client only — existing clients already have EFT enabled. Runs early in Initial Onboarding so the owner can make their initial contribution without delay.

## Procedure

Very important to enable EFT Payments to allow Owners to make the initial contribution.

Buildium → Owner Profile → Financials Tab → Contributions ePay Settings → Edit → Check **EFT**.

Repeat for all Owners on the Account.
