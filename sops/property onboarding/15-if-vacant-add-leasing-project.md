## When This Runs

Runs during Initial Onboarding for any property that is vacant at onboarding. Creates the Leasing 2.0 task that kicks off Turnover and/or Lease-Up work. Skip for fully occupied properties.

## Procedure

Create Asana Task:

* **Task Template:** `(New Property) Leasing | <address>`
* **Project:** Leasing 2.0
* **Section:** To Be Assigned
* **Assigned to:** Courtney
* **Due:** Same or next day
* Fill in required fields: **Unit ID**, **Special Note:** NEW PROPERTY
* Upload Owner Onboarding Form so team has context
* Leave comment providing any relevant notes / special instruction
* Use subtasks for relevance: **Move Out**, **Lease Up**, **Turn Over**
* **Process Deposit Subtask** — Mark complete only if there is no deposit to be refunded. If owner would like to refund the security deposit themselves: make note in comment, assign per usual.
