## When This Runs

During Initial Onboarding once the unit(s) exist in Buildium. Captures owner-selected rental policies and listing context so the leasing team has what it needs when the unit goes to market.

## Procedure

Add any notes or small details about the unit that would be helpful to the leasing team — minor details they could easily miss. E.g. "Lower, daylight basement unit. Mini split for heat & A/C."

* Prep listing with owner-selected Rental Policies
* Copy / paste Rental Policy Template
* If owner is unsure on Pet Policy — input our standard policy: **Up to 2 pets, 30lb weight limit, $50/mo pet rent**
* Accounting team will populate the Utility portion — use **"tenant responsible for all utilities"** as the default
* Copy / paste "Any other features to describe in the listing?"
* Edit Amenities — if pet friendly, be sure to check off **Cat**, **Dog**, and **Pet Friendly** options
