## When This Runs

Runs during Initial Onboarding if the property is part of a Homeowners Association. Establishes contact with the HOA, captures rules/fees/amenities, and updates the HOA Playbook for downstream teams. Skip for properties with no HOA.

## Procedure

* Send Intro email to HOA contact
* Follow up every 3-4 days until response is received
* Once received, update [HOA Playbook](../inspections/hoa-playbook.md) with details
* Create subfolder in HOA Docs Shared File → Upload any relevant docs → Link to playbook
* Update Unit Listing with any confirmed amenities that are unlisted
