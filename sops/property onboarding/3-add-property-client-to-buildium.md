## When This Runs

Once the Onboarding Form is on the Asana task, create the property and rental owner records in Buildium. This establishes the canonical property ID used by every downstream system (Asana sync, billing, leasing, maintenance).

## Procedure

Rentals → Properties → Add Property → Fill in boxes / Copy / paste Property & Client details from the Onboarding Form to Buildium.

* **Property Type** — from Onboarding Form.
* **Address** — from Onboarding Form.
* **Add Rental Owner** — Copy / paste all owners listed on the Onboarding Form. Equal ownership split — Accounting will confirm later. *(Adding Property: choose existing owner rather than creating new.)*
* **Operating Account** — Choose "Temp - Onboarding".
* **Property Reserve** — Usually $1,000 → Confirm amount on PM Agreement.
* **Enter Units** — Label actual unit numbers if available. Bed / bath / SQFT — obtain from Zillow / Redfin. Add another unit & repeat, if applicable.
* Click **Create Property**.
