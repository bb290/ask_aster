## When This Runs

Update the AUM Google Map so the property's marker reflects the new unit count, and move the property between Single Family and MultiFamily tables if the unit addition changes its classification.

## Procedure

* [Sagareus AUM Google Map](https://www.google.com/maps/d/u/0/edit?mid=1nVYVXbnQqzOlV17EFVaY1skcj7Gg98w&ll=47.52830654532093%2C-122.4681203&z=9)
* Click 3 dots next to MultiFamily Properties or Single Family Homes, as appropriate → Open Data Table → Search property address
* If property is still multi-family → Edit Unit Count Only
* If single family converting to multi-family → Delete row from Single Family Data Table → Add row to Multifamily Data Table
* Property should populate onto map
