## When This Runs

During Initial Onboarding, after the property is in Buildium. Creates the 1Password entry that holds keybox codes and key-management details for this property going forward.

## Procedure

1Password → Vaults → **Property Access** → View Vault → **00 Keybox** template → 3 dots → **Duplicate** → Edit → Rename Note to **Subject Property Address**.
