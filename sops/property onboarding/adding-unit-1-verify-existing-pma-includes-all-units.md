## When This Runs

Before doing anything else in the Adding Unit workflow, confirm the existing Property Management Agreement covers the unit being added. If the PMA is scoped to a specific unit, a PMA revision must be completed before proceeding.

## Procedure

* Locate existing PM Agreement → Ensure the address does **not** specify just 1 unit:
  * 5521 Brooklyn Ave Seattle WA → This is general enough to include all units
  * 5521 Brooklyn Ave #A Seattle WA → If we are adding Unit B, we need to revise the PM Agreement
