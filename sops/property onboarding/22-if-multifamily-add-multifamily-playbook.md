## When This Runs

Runs during the Messy Middle phase for multifamily properties only. Adds the property to the Multifamily Playbook and spawns a Parking Audit task in Leasing 2.0 to inventory parking and storage assignments. Skip for single-family homes.

## Procedure

### CREATE | Parking Audit Task

Multi-Family Only. Create Asana Task:

* **Project:** Leasing 2.0
* **Task Template:** "Parking Audit"
* **Section:** Parking + Storage
* **Assigned to:** Leasing Assistant
* **Due:** Same or next day
