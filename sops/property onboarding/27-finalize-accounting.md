## When This Runs

Runs during the Finalize Onboarding stage after W-9 and EFT details are entered. Assigned to the Accounting Team Lead, who double-checks all financial inputs, sets up owner reporting files and bank accounts, sends the test draw, and sends initial owner reports. This is a KPI-triggering step — the **Initial Reports Sent** date is recorded here and drives the Growth Dashboard metric.

## Procedure

Once Accounting Form Details have been input → Assign Finalize Account to Accounting Team Lead. Accounting Team Lead double checks all financial inputs & sets up owner reporting file & bank accounts.

### ASSIGN | Referral Commission

If applicable, assign referral commission in accounting system.

### VERIFY | Owner Profile Details

* Mgmt Fees
* EFT
* W-9

### VERIFY | Property Settings Fields

Verify all Property Settings fields are accurate.

### CONFIGURE | Owner Google Drive File

Navigate to Google Drive → Search **"Active Portfolio"** → Select File → Select **"Onboards"** File → Create New File → Title: `[Account Number] [Owner Last name]`

### ASSIGN | Buildium Bank Account

* Review all bank accounts in Buildium (Select **Accounting → Banking → View All**)
* Review unused and available accounts — Labeled in 900s
* Select the available account → Rename to a unique number that is available with the **201-799** labels → Include unique number and last name of owner
* Navigate back to subject property profile → Select **Edit** of Banking details tile → **Operating** set to the subject available and renamed account → **Security deposit** set to **9990** account

### UPDATE | Bank Account Nickname

* **In Bank of America:** Navigate to subject account → Select the Pencil Icon next to **Account Name** → Input the Account Number and Last Name to match Buildium and Owner File
* **In Chase:** Select any account → Select **More** → Select **"Update settings and preferences"** → Select **Account Nickname** → Navigate to subject account by search last four → Input Account Number and Last Name to match Buildium and Owner File → Select **Save Changes**

### PREPARE | Initial Reports

Begin Owner Report at January 1st of the year where income was first received. Reconcile and include all reports detailed in the owner reporting section.

### SEND | Test Draw

Initiate $17 owner draw to the owner's bank account. **Shortcut menu → Owner Draw by EFT** → Enter subject address → Manually Adjust the amount to **$17** → **Memo:** Owner Draw → Submit.

### SEND | Initial Owner Reports & UPDATE | Initial Reports Sent

* Locate owner file in the **"Onboards"** file of Active Portfolio Parent Folder
* Upload the initial owner reports to the file
* Click **Share** → Adjust settings to **anyone with the link** → Copy Link
* Draft email to owner → Hyperlink the Google Drive file to the **HERE** section → Send
* **Update the Initial Reports Sent date field** on the parent task
