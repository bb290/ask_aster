## Escalation


:::warning
* **No reply by deadline:**
  * Review recent move ins in Rent Collection
* **Still unresolved after escalation:**
  * Assign resulting task with context to accounting manager

:::


---

## DO NOT DEPOSIT


:::warning
* When checks are received via mail and the payment does not fully settle the balance due.
* Consult the running Asana Task titled
  * Collection // Running task // Do not deposit list
* Confirm if any checks received are from a tenant on the list.
  * If not, deposit
  * If so, do not deposit and set aside in envelope.
* Assign return check task to lead accountant and indicate where the envelope is located. Use a subtask to assign on the parent task.
* Lead accountant sends the check to the tenants address via certified mail each Tuesday.

:::


---


:::tip
**Summary:** When a bank deposit has no matching Buildium transaction, create an Asana task, email the leasing team for identification, and escalate if unresolved. Never deposit checks that do not fully settle a balance due — consult the do-not-deposit list first.

:::
