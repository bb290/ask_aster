## Business License Side (DOR MyDOR)

When closing a Burien property, determine whether the owner retains other Burien or WA-state properties using the same DOR biz license. If sole Burien property, proceed with DOR close-out via MyDOR (see [Registration & License // Kirkland // Cancel](registration-license-kirkland-cancel.md)). Otherwise, the license and endorsement stay active; only the property-specific entry is removed.
